# (Nouvelle interface utilisateur) Renommer un portfolio

*Fonction*

Renommez rapidement un portfolio sans ouvrir les paramètres complets du portfolio.

1. Dans le menu principal, cliquez sur **[!UICONTROL Manage]>[!UICONTROL Portfolios]**.

1. Placez le curseur sur la ligne du portfolio, puis cliquez sur **[!UICONTROL ...]>[!UICONTROL Rename]**.

1. Modifiez le nom, puis cliquez sur **[!UICONTROL Apply]**.
