# (Nouvelle interface utilisateur) Afficher l’historique des modifications d’un portefeuille

*Fonction*

Vous pouvez afficher un journal des modifications apportées au portefeuille au cours d’une période. Les actions consignées incluent la création du portefeuille, toutes les modifications apportées aux paramètres du portefeuille, toutes les campagnes affectées au portefeuille ou non affectées à celui-ci, toutes les contraintes appliquées aux unités d&#39;offre du portefeuille ou supprimées de celles-ci, toutes les contraintes sur les unités d&#39;offre du portefeuille qui ont expiré et d&#39;autres actions qui affectent le portefeuille

Vous pouvez filtrer les données par catégorie<!-- Not available as of 5-14: and by the user who performed the action -->. Vous pouvez également trier la liste par date dans l’ordre croissant ou décroissant.

Vous pouvez également télécharger les données dans un fichier de feuille de calcul.

## (Nouvelle interface utilisateur) Afficher l’historique des modifications d’un portefeuille {#portfolio-change-history}

Les informations de chaque modification incluent la date, le nom d’utilisateur de la personne qui a effectué la modification, le type de modification et la valeur modifiée.

1. Dans le menu principal, cliquez sur **[!UICONTROL Goals]>[!UICONTROL Conversions]**.

1. Placez le curseur sur la ligne du portfolio, puis cliquez sur **[!UICONTROL ...]>[!UICONTROL Change History]**.

1. (Facultatif) Modifiez les informations affichées :

   * Cliquez sur ![Calendrier](/help/search-social-commerce/assets/calendar-new.png "Calendrier") pour modifier la période du rapport.

   * Cliquez sur ![Filtrer](/help/search-social-commerce/assets/filter-new.png "Filtrer") pour filtrer les données par type de modification.<!-- Not available as of 5-14: and by the user who performed the action -->.

   * Cliquez sur ![Trier](/help/search-social-commerce/assets/sort.png "Trier") pour trier la liste par date dans l’ordre croissant ou décroissant.

## Gérer les rapports de données de performances pour les historiques de modifications

Vous pouvez télécharger des informations supplémentaires sur l’historique des modifications d’un portfolio dans un fichier [!DNL Microsoft Excel] format de classeur (fichier XLSX). Le rapport comprend l’identifiant de modification numérique, les anciennes et les nouvelles valeurs, le type de modification (par exemple « ajouté » ou « modifié »), le type d’action (par exemple « Budget » ou « Nom du Portfolio ») et le sous-type, le cas échéant, l’heure et le nom d’utilisateur.

### Générer un rapport avec les lignes de données filtrées

1. [Ouvrez l’historique des modifications du portefeuille](#portfolio-change-history).

1. Dans le coin supérieur droit, cliquez sur ![Télécharger le rapport](/help/search-social-commerce/assets/download.png "Télécharger le rapport").

1. Dans les paramètres de [!UICONTROL Grid Reports], saisissez un nom de rapport unique, puis cliquez sur **[!UICONTROL Generate]**.

   Par défaut, le fichier est nommé « portfolioChangeHistory_YYYYMMDD_NNNN », où « NNNN » correspond au numéro de tâche séquentiel (par exemple, « portfolioChangeHistory_20260402_1326 »).

   Le fichier est ajouté à la liste de [!UICONTROL Recently Generated].

1. (Facultatif) Pour télécharger le fichier une fois l’opération terminée, cliquez sur ![Télécharger](/help/search-social-commerce/assets/download.png "Télécharger") en regard du nom du fichier.

   Le fichier est téléchargé conformément à la procédure normale de votre navigateur.

### Télécharger un rapport terminé

1. [Ouvrez l’historique des modifications du portefeuille](#portfolio-change-history).

1. Dans le coin supérieur droit, cliquez sur ![Télécharger le rapport](/help/search-social-commerce/assets/download.png "Télécharger le rapport").

1. Dans la liste [!UICONTROL Recently Generated] de la boîte de dialogue [!UICONTROL Grid Reports], cliquez sur ![Télécharger](/help/search-social-commerce/assets/download.png "Télécharger") en regard du nom du fichier.

   Le fichier est téléchargé conformément à la procédure normale de votre navigateur.

### Supprimer un rapport terminé

1. [Ouvrez l’historique des modifications du portefeuille](#portfolio-change-history).

1. Dans le coin supérieur droit, cliquez sur ![Télécharger le rapport](/help/search-social-commerce/assets/download.png "Télécharger le rapport").

1. Dans la liste [!UICONTROL Recently Generated] de la boîte de dialogue [!UICONTROL Grid Reports], cliquez sur ![Supprimer](/help/search-social-commerce/assets/delete-new.png "Supprimer") en regard du nom du fichier.
