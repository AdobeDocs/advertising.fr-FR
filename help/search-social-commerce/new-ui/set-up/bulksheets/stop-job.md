# (Nouvelle interface utilisateur) Arrêter une tâche de feuille d’envoi groupé en cours

Vous pouvez arrêter n&#39;importe quelle tâche de feuille d&#39;envoi groupé, y compris un fichier en cours de publication, pendant qu&#39;elle est toujours en cours. Vous pouvez également arrêter des tâches planifiées mais qui n’ont pas encore commencé.

1. Dans le menu principal, cliquez sur **[!UICONTROL Setup]** \> **[!UICONTROL Bulksheets]**.

1. Cochez la case en regard de chaque tâche à arrêter.

1. Dans la barre d’actions, cliquez sur **[!UICONTROL Stop Job]**.

1. Dans le message de confirmation, cliquez sur **[!UICONTROL Confirm]**.

>[!NOTE]
>
>Si vous annulez la validation d&#39;un fichier de feuille d&#39;envoi groupé après la validation de certaines lignes, une feuille d&#39;envoi groupé Erreurs EF est générée avec les lignes non validées.
