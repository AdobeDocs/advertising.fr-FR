# (Nouvelle interface utilisateur) À propos de la vue [!UICONTROL Keywords]

*Fonction Beta*

La vue [!UICONTROL Manage] > [!UICONTROL Keywords] comprend deux onglets :

* L’onglet [!UICONTROL Keywords] répertorie tous les mots-clés de la vue filtrée pour le compte d’annonceur sélectionné.

* L’onglet [!UICONTROL Negatives] répertorie tous les mots-clés négatifs dans la vue filtrée pour le compte d’annonceur sélectionné.
