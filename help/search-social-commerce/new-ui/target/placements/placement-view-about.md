# (Nouvelle interface utilisateur) À propos de la vue [!UICONTROL Placements]

*Fonction Beta*

La vue [!UICONTROL Manage] > [!UICONTROL Placements] comprend deux onglets :

* L’onglet [!UICONTROL Placements] répertorie tous les emplacements dans la vue filtrée pour le compte d’annonceur sélectionné.

* L’onglet [!UICONTROL Negatives] répertorie tous les emplacements négatifs dans la vue filtrée pour le compte d’annonceur sélectionné.
