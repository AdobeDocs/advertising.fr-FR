# [!UICONTROL RSA Asset Report]

*[!DNL Google Ads]des comptes avec des annonces responsive sur le Réseau de Recherche uniquement*

Le [!UICONTROL RSA Asset Report] comprend des mesures d’impression quotidiennes pour chaque ressource ([!UICONTROL Creative Title] ou [!UICONTROL Description]) pour les Responsive Search Ads (RSA) dans un ou plusieurs portfolios ou un ou plusieurs comptes, campagnes et groupes publicitaires. Par défaut, les données incluent une ligne pour chaque ressource qui a reçu au moins une impression dans la période spécifiée, même si la ressource a été désactivée (supprimée) pendant la période couverte par le rapport. Par défaut, les lignes sont classées par groupe publicitaire dans l’ordre croissant. Le rapport peut éventuellement inclure des classifications de libellés.

>[!NOTE]
>
>Lorsque vous sélectionnez l’option « [!UICONTROL Include rows with no performance data] », les résultats n’incluent toujours pas les données de ressources qui n’ont jamais reçu de données.

Vous pouvez afficher les données à partir du 8 août 2022.<!-- Later: You can view data for the previous 36 months. -->
