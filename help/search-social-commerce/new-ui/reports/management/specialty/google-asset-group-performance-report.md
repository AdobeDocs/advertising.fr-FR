# [!UICONTROL Google Asset Group Performance Report]

*Applicable uniquement aux comptes [!DNL Google Ads] avec des campagnes Performance Max*

Le [!UICONTROL Google Asset Group Performance Report] inclut des données de performances pour les groupes de ressources dans les campagnes Performance Max au sein d’un compte spécifié. L’état inclut le statut principal et les raisons du statut pour chaque groupe de ressources inclus. Par défaut, les données incluent une ligne pour chaque groupe de ressources ayant reçu au moins une impression dans la période spécifiée, même si le groupe de ressources a été supprimé pendant la période couverte par le rapport. Par défaut, les lignes sont classées par ordre croissant dans les campagnes.
