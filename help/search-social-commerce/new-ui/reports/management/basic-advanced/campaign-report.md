# [!UICONTROL Campaign Report]

La [!UICONTROL Campaign Report] inclut les données de coût, de clic et (éventuellement) de conversion pour une ou plusieurs campagnes avec des mots-clés. Par défaut, les données incluent une ligne pour chaque campagne applicable pour chaque unité de temps dans la période spécifiée. Les lignes sont dans l’ordre croissant, d’abord par campagne, puis (le cas échéant) par portefeuille.

Vous pouvez éventuellement inclure des classifications de libellés spécifiques à l’annonceur.

Vous pouvez afficher les données des 36 mois précédents.
