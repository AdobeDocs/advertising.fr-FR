# [!UICONTROL Transaction Report]

Le [!UICONTROL Transaction Report] fournit des détails au niveau de la transaction pour les mots-clés actuels dans les groupes publicitaires dans la recherche et
affichez les campagnes (et pour les publicités dans les campagnes sociales) dans des portfolios ou des comptes publicitaires spécifiés (le cas échéant). Vous pouvez éventuellement inclure des mesures de conversion. Vous pouvez comparer les résultats de ce rapport aux données de transaction de l’annonceur afin de vérifier que Search, Social et Commerce signalent toutes les transactions de l’annonceur.

Par défaut, le rapport inclut des données pour les transactions qui se sont produites au cours de la période spécifiée. Les données de chaque transaction incluent l’identifiant de transaction, les heures de clic/impression et de transaction, le type de conversion, la mesure de conversion et les données de chiffre d’affaires pour la mesure de conversion (par exemple, 1 pour un enregistrement ou 12 pour une commande de 12 $) pour chaque mot-clé ou combinaison de mot-clé ou de groupe publicitaire/campagne/portefeuille. Par défaut, les lignes sont classées par ordre croissant, d’abord par heure de transaction, puis par heure de clic.

Vous pouvez afficher les données des 45 jours précédents.

>[!NOTE]
>
>* Si plusieurs unités d’enchères avec différents types de correspondance ont le même ID de transaction, le chiffre d’affaires de l’ID de suivi est fractionné en fonction du nombre de clics effectués à la date de clic spécifiée.
>* Si le rapport inclut des données provenant de groupes publicitaires dans des campagnes de recherche activées pour le contenu, dans des campagnes d’affichage ou dans des campagnes sociales (qui n’incluent pas de mots-clés), la colonne [!UICONTROL Keyword] du rapport terminé inclut les noms de groupe publicitaire applicables, tels que « (contenu du groupe publicitaire) Votre nom de groupe publicitaire ».
