# [!UICONTROL Constraint Report]

Le [!UICONTROL Constraint Report] inclut des données de coût, de clic et (éventuellement) de conversion pour les contraintes qui utilisent l’architecture de classification de libellés, agrégées dans les portfolios, les réseaux publicitaires, les comptes, les campagnes ou les groupes publicitaires. Par défaut, les données incluent une ligne pour chaque contrainte applicable pour chaque unité de temps dans la période spécifiée. Les lignes sont dans l’ordre croissant, d’abord par contrainte, puis par date de début pour l’unité de temps par défaut.

Vous pouvez afficher les données des 36 mois précédents.

>[!NOTE]
>
>Toute modification apportée aux contraintes d’une entité est visible en une heure environ.
