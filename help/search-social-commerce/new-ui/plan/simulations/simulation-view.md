# Afficher les détails de la simulation

*Fonction Beta*

Vous pouvez afficher les détails de la simulation à l’écran sous la forme d’un graphique ou d’un tableau.

1. Dans le menu principal, cliquez sur **[!UICONTROL Plan]>[!UICONTROL Simulations]**.

1. Cliquez sur le nom des simulations.

   La simulation s’ouvre dans la vue graphique.

1. (Facultatif) Dans l’affichage graphique, effectuez l’une des opérations suivantes :

   * Pour modifier la mesure pour laquelle les données estimées sont affichées pour chaque niveau de cible des dépenses (étape), sélectionnez une nouvelle mesure dans le menu **[!UICONTROL Y-Axis Metric]**.

   * (Simulations hebdomadaires uniquement) Saisissez une nouvelle valeur de **[!UICONTROL Target Mid Point]** pour afficher les données prévues à l’aide de cette valeur.

   * (Simulations hebdomadaires uniquement) Dans le menu **[!UICONTROL Constraints]**, choisissez si vous souhaitez afficher les *[!UICONTROL With Constraints]* de données, les *[!UICONTROL Without Constraints]* ou les *[!UICONTROL With and Without]*.

1. (Facultatif) Pour ouvrir la vue Tableau, cliquez sur l’onglet **[!UICONTROL Table View]** .

1. (Facultatif) Pour afficher les paramètres en lecture seule du portfolio applicable, cliquez sur **[!UICONTROL Portfolio Settings]** dans le coin supérieur droit.
