# [!DNL Microsoft Advertising] les paramètres de publicité du produit

Le corps de l’annonce publicitaire est créé automatiquement à partir des informations sur les produits dans [!DNL Microsoft Merchant Center], en fonction des groupes de produits ciblés pour le groupe publicitaire. Vous pouvez éventuellement créer des lignes de promotion à inclure dans les annonces de produits des campagnes qui utilisent le réseau d’achat.

Pour plus d’informations sur les publicités de produits [!DNL Microsoft Advertising], consultez la [documentation d’Advertising Microsoft](https://help.ads.microsoft.com/#apex/3/en/51082).

## [!UICONTROL Product Ad]

**[!UICONTROL Promotion Line]:** (facultatif) Ligne de promotion unique à inclure dans la liste de produits dans les résultats de recherche (par exemple, « Livraison gratuite !). La longueur maximale est de 45 caractères.

La ligne de promotion peut apparaître à différents emplacements par rapport à l’annonce (comme sous l’annonce), selon l’emplacement de l’annonce sur la page.
