# [!UICONTROL Google Performance Max Placement Report]

*Applicable uniquement aux comptes [!DNL Google Ads] avec des campagnes Performance Max*

Le [!UICONTROL Google Performance Max Placement Report] inclut des données d’impression pour les emplacements dans des campagnes [!DNL Google Ads] Performance Max au sein d’un compte spécifié. Le rapport inclut les impressions pour chaque emplacement inclus par URL cible. Par défaut, les données incluent une ligne pour chaque combinaison d’emplacement et d’URL cible qui a reçu au moins une impression dans la plage de données spécifiée, même si l’emplacement a été supprimé pendant la période de création de rapports. Par défaut, les lignes sont classées par ordre croissant dans les campagnes.

Utilisez ce rapport pour rechercher les URL à exclure et améliorer les performances des campagnes Performance Max.
