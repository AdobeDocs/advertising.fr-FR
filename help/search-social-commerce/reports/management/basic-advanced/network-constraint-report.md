# [!UICONTROL Network Constraint Report]

La [!UICONTROL Network Constraint Report] affiche tous les budgets de réseau publicitaire minimum et maximum spécifiés ainsi que les dépenses réelles (coût). Le rapport inclut des données de coût, de clic, d’impression, d’affichage publicitaire et (éventuellement) de conversion pour un ou plusieurs portefeuilles. Par défaut, les données incluent une ligne pour chaque compte de réseau publicitaire applicable dans la période spécifiée. Les lignes sont dans l’ordre croissant, d’abord par portfolio, puis par réseau publicitaire.

Les données sont disponibles à partir du 15 octobre 2022.<!-- [Later: You can view data for the previous NN days.] -->
