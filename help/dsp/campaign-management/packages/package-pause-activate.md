# Mettre en pause ou activer un package

Mettre en pause un package actif ou activer un package en pause lorsqu’il se trouve dans le planning de vol configuré.

Tous les emplacements suspendus dans un package avant de mettre le package en pause lui-même restent suspendus lorsque le package est activé.

## Mettre en pause un package

1. Dans le menu principal, cliquez sur **[!UICONTROL Campaigns]**.
1. Cliquez sur le nom de la campagne pour ouvrir la vue [!UICONTROL Packages].
1. En regard du nom du package, cliquez sur **[!UICONTROL ...]** > **[!UICONTROL Pause]**.

>[!NOTE]
>
>La suspension d’un package met en pause tous les emplacements, y compris les emplacements programmatiques garantis (PG) dans le package. Contactez les éditeurs concernés si les enchères entrantes doivent être suspendues.

## Activer un package

1. Dans le menu principal, cliquez sur **[!UICONTROL Campaigns]**.
1. Cliquez sur le nom de la campagne pour ouvrir la vue [!UICONTROL Packages].
1. En regard du nom du package, cliquez sur **[!UICONTROL ...]** > **[!UICONTROL Activate]**.
