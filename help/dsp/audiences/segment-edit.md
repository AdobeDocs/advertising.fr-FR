# Modifier les informations sur le segment

Vous pouvez modifier le nom et l’intervalle de recherche en amont d’un segment personnalisé, ou le nom d’un segment d’exclusion de vente du CCPA. Les balises de segment ne changent pas lorsque vous modifiez les paramètres.

1. Dans le menu principal, cliquez sur **[!UICONTROL Audiences]** > **[!UICONTROL Segments]**.

1. Placez le curseur sur la ligne de segment et cliquez sur **[!UICONTROL Edit]**.

1. Modifiez les informations sur le segment :

   1. (Facultatif) Modifiez le **[!UICONTROL Name]** de segment.

      Chaque nom de segment doit être unique pour le type de segment.

   1. (Segments personnalisés uniquement ; facultatif) Modifiez le **[!UICONTROL Segment Window]**, qui correspond au nombre de jours pendant lesquels le cookie d’un utilisateur reste dans le segment.

      Entrez une valeur comprise entre 1 et 365.

   1. Cliquez sur **[!UICONTROL Save]**.
