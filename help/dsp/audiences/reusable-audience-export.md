# Exporter les détails d’une audience réutilisable

Vous pouvez télécharger un fichier de feuille de calcul [!DNL Microsoft Excel] avec la règle de configuration de l’audience, le nombre de membres actifs de l’audience par type d’appareil, les détails de chaque segment de l’audience et tous les emplacements qui ciblent l’audience.

1. Dans le menu principal, cliquez sur **[!UICONTROL Audiences]** > **[!UICONTROL All audiences]**.

1. Placez le curseur sur la ligne d’audience et cliquez sur **[!UICONTROL Export XLS]**.

   Le fichier est téléchargé au format XLSX conformément à la procédure normale de votre navigateur. Le nom de fichier comprend le nom et la date de l’audience.
