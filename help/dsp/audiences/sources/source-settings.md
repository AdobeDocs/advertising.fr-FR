# Paramètres de la source d’audience

*Fonction bêta*

**[!UICONTROL Data Visibility Level]:** Si les segments sont disponibles pour un seul annonceur ayant accès au compte (*[!UICONTROL Advertiser]*) ou à tous les annonceurs ayant accès au compte *[!UICONTROL Account]*.

**[!UICONTROL Advertiser]:** (Visibilité au niveau du annonceur uniquement) Annonceur pour lequel les segments sont disponibles. Sélectionnez-en un dans la liste des annonceurs ayant accès au compte.

**[!UICONTROL Enter IMS Org Id]:** L’ID d’organisation Experience Cloud pour la variable [!DNL Adobe Experience Platform] compte .

**[!UICONTROL AAC Source Key]:** (Lecture seule) ; généré lorsqu’une valeur est saisie dans la variable [!UICONTROL Enter IMS Org Id] (Champ) Clé source que vous pouvez utiliser pour [créer une connexion de destination dans [!DNL Adobe Real-Time Customer Data Profile]](https://experienceleague.adobe.com/docs/experience-platform/destinations/catalog/advertising/adobe-advertising-connection.html) pour envoyer des audiences vers le DSP de publicité. Vous pouvez copier la valeur dans le presse-papiers pour la coller dans le [!DNL Adobe Real-Time CDP] paramètres de connexion à la destination ou dans un fichier .
