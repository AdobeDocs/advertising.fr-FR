# Utilisation de l’inventaire des [!DNL Roku]

Advertising DSP offre des fonctionnalités de publicité sur [!DNL Roku].

## Correspondance d’audience

Le partenariat [!DNL Roku] et DSP associe vos audiences [!DNL DSP] à des identifiants [!DNL Roku] pour un ciblage d’audience déterministe 1:1 sur [!DNL Roku] inventaire.

## [!DNL Roku] options d&#39;inventaire

Vous pouvez effectuer l’une des opérations suivantes :

* Configurez les ID d’offres privées directement avec [!DNL Roku], puis saisissez les données d’ID d’offres dans DSP.

* Visitez la galerie [!DNL On Demand] pour vous abonner aux profils [!DNL Roku] :

>[!NOTE]
>
>[!DNL Roku] stock n&#39;est pas disponible sur les marchés ouverts et les bourses.

* Pour vos offres privées, [configurez des informations sur les ID d’offres dans DSP](/help/dsp/inventory/deal-id-create.md) puis ciblez « [!UICONTROL Roku Network - Audience] » et « [!UICONTROL The Roku Channel - Audience] » dans [!DNL Roku] emplacements.<!-- Or do you target the deal ID?? * Vous pouvez [vous abonner à l’inventaire suivant [!DNL Roku] dans la  [!DNL On Demand] galerie](/help/dsp/inventory/on-demand-inventory-subscribe.md), puis cibler l’une des offres approuvées dans [!DNL Roku] emplacements :

   * « [!UICONTROL Roku Network - Audience] » pour l’inventaire dans l’écosystème [!DNL Roku] avec des partenaires de contenu premium, tels que [!DNL The CW], [!DNL ABC] et [!DNL ESPN].

   * « [!UICONTROL The Roku Channel - Audience] » pour [!DNL Roku] contenu d’application détenu et exploité.

### Avantages de la personnalisation des places de marché privées avec [!DNL Roku]

Les offres privées vous permettent de personnaliser les paramètres de l&#39;offre selon vos besoins.

* **Tarification négociée :** travaillez avec l’équipe des ventes [!DNL Roku] pour négocier et structurer une transaction qui répond à vos besoins de campagne.

* **Priorité d’échelle :** les marchés privés (PMP) bénéficient d’une priorité supérieure aux offres permanentes (telles que les offres [!DNL On Demand]).

* **Gestion des fréquences :** la limite de fréquence par défaut [!DNL Roku] est d’une (1) annonce par tranche de 30 minutes par utilisateur, mais vous pouvez la personnaliser par heure, jour, semaine, mois ou période de vol complète.<!-- Within the DSP placement settings? NO - you negotiate this with Roku, but Christine to confirm with Amanda whether you should be able to edit this in placement. -->

* Ciblage des données **[!DNL Roku]:** audiences [!DNL Roku] sont créées à partir de signaux d’appareil et de télévision [!DNL Roku], de données suivies par [!DNL The Roku Channel] (comme l’affinité genre TV, le comportement de la télévision en flux continu et le statut d’abonnement au câble) et de données supplémentaires provenant du système de gestion de la relation client (CRM) [!DNL Roku].

* **[!DNL Roku]le ciblage de contenu** les offres privées peuvent cibler les applications par genre, application de la place sur la liste bloquée de l’application, événements saisonniers et de tentpole, et spectacles en [!DNL The Roku Channel] uniquement.

## [!DNL Roku] des emplacements

Dans les campagnes DSP, [créez [!DNL Roku] des emplacements spécifiques](/help/dsp/campaign-management/placements/placement-create.md) en utilisant le type d’emplacement « [!UICONTROL Connected TV (Roku)] ». Incluez des emplacements [!DNL Roku] dans des packages spécifiques à [!DNL Roku] avec des objectifs définis.

Chaque emplacement [!DNL Roku] doit cibler au moins une offre ou une source [!DNL Roku]. Pour utiliser la correspondance d’audience DSP avec [!DNL Roku], incluez un ou plusieurs segments d’audience qui peuvent être associés au jeu de données déterministe [!DNL Roku] (accepté).

### Fournisseurs de suivi tiers approuvés par le [!DNL Roku]

[!DNL Roku] emplacements peuvent inclure des pixels d’événement tiers et des pixels de conversion des fournisseurs suivants : [!DNL Acxiom], [!DNL Comscore], [!DNL Data Plus Math], [!DNL Experian], [!DNL Factual], [!DNL Kantar], [!DNL Marketing Evolution], [!DNL Neustar], [!DNL Nielsen], [!DNL Nielsen Catalina Solutions], [!DNL NinthDecimal], [!DNL Oracle], [!DNL Placed], [!DNL Polk], [!DNL Research Now], et.

### Bonnes pratiques par stratégie de placement

Vous trouverez ci-dessous les pratiques recommandées pour les emplacements spécifiques à [!DNL Roku].

Pour maximiser la portée incrémentielle :

* Supprimez les audiences exposées sur [!DNL Roku O&O] en excluant les audiences que vous avez déjà atteintes à l’aide de [!DNL The Roku Channel].

* Supprimez les audiences exposées sur [!DNL All Roku] en excluant les audiences que vous avez déjà atteintes sur la plateforme [!DNL Roku].

Pour une configuration plus rapide :

* Ciblez les offres existantes et permanentes pour les [!DNL The Roku Channel] dans [[!DNL On Demand] Inventaire](/help/dsp/inventory/on-demand-inventory-subscribe.md) afin d&#39;accéder rapidement [!DNL Roku] stock détenu et exploité.
* Ciblez les offres existantes et permanentes pour les [!DNL Roku Network] dans [[!DNL On Demand] Inventaire](/help/dsp/inventory/on-demand-inventory-subscribe.md) afin de les mettre rapidement à l’échelle sur la plateforme [!DNL Roku].

À l’échelle maximale :

* Personnalisez un marché privé [!DNL Roku] pour un accès prioritaire à [!DNL Roku]’approvisionnement à un prix négocié.
