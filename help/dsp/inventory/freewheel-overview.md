# Présentation de la configuration d’offres programmatiques garanties dans [!DNL Freewheel]

La configuration d’offres programmatiques garanties avec les éditeurs sur [!DNL Freewheel] nécessite des autorisations et des étapes supplémentaires.

>[!PREREQUISITES]
>
>Contactez l’équipe chargée de votre compte Adobe pour vous assurer que votre compte [!DNL DSP] dispose des autorisations suivantes :
>
>* Autorisation d’utiliser le workflow programmatique garanti [!DNL FreeWheel], qui est requis pour envoyer une annonce pour une offre programmatique garantie à [!DNL FreeWheel].
>
>* (Si vous travaillez avec des éditeurs britanniques qui ont besoin d’un numéro d’horloge [!DNL Clearcast] pour chaque publicité) Autorisation d’inclure des numéros d’horloge dans vos publicités.

## Workflow

1. Créez une annonce publicitaire avec le type de média spécifié dans l’offre.

   Pour certains éditeurs britanniques, vous devez inclure un numéro d&#39;horloge [!DNL Clearcast] avec votre publicité.

1. [Acceptez l’ID d’offre](#programmatic-guaranteed-set-up.md#pg-setup-deal-id-inbox) que vous avez déjà négocié avec un éditeur sur [!DNL Freewheel] à l’aide de la boîte de réception d’ID d’offre.

   Après avoir accepté l’offre, suivez les invites pour 1) sélectionner l’annonce à utiliser pour l’offre et 2) créer un emplacement par défaut programmatique garanti pour diffuser l’annonce.

1. [Envoyez l’annonce à  [!DNL Freewheel]](freewheel-submit.md)

   La publicité doit être soumise et approuvée avant son exécution.

1. [Vérifiez le statut d’envoi de l’annonce](freewheel-check-status.md).
