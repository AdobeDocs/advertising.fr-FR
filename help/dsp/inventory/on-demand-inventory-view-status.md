# Consulter le statut de vos demandes d’offres [!DNL On Demand] et de vos abonnements

Une fois que vous avez demandé une offre individuelle ou que vous vous êtes abonné à toutes les offres d’un éditeur, vérifiez si l’éditeur a approuvé ou rejeté chaque offre pour chaque annonceur de votre compte.

1. Dans le menu principal, cliquez sur **[!UICONTROL Inventory]** > **[!UICONTROL On Demand]**.
1. Filtrez les offres disponibles pour inclure les offres que vous avez demandées ou auxquelles vous êtes abonné (**[!UICONTROL Currently subscribed to]**)**.
1. Effectuez l’une des opérations suivantes :
   * Pour afficher les offres dans la vue [!UICONTROL Deal], cliquez sur **[!UICONTROL Deal view]**.
   * Pour afficher les opérations dans la vue [!UICONTROL Subscription] :
      1. Cliquez sur **[!UICONTROL Subscription view]**.
      1. Placez le curseur sur le logo de l’éditeur, puis cliquez sur **[!UICONTROL See Deals]**.
1. Recherchez la colonne [!UICONTROL Status] pour afficher le statut d’approbation.

## Statut de l’affaire

L’approbation ou le refus de l’éditeur peut être basé sur le type de compte et la catégorie de campagne pour le compte, la catégorie d’annonceurs et la liste des annonceurs bloqués de l’éditeur.

* ![entièrement approuvé](/help/dsp/assets/approved.png) pour les demandes d’offre qui ont été approuvées pour tous les annonceurs du compte

* ![partiellement approuvé](/help/dsp/assets/partly-approved.png) pour les demandes d’offre qui ont été approuvées pour certains annonceurs du compte, mais pas pour tous

  Pour savoir pourquoi une demande a été refusée pour un annonceur, cliquez sur ![partiellement approuvé](/help/dsp/assets/partly-approved.png).

* ![refusé](/help/dsp/assets/denied.png) pour les demandes d’offre qui ont été refusées pour tous les annonceurs du compte

  Pour savoir pourquoi une demande a été refusée pour chaque annonceur, cliquez sur ![refusé](/help/dsp/assets/denied.png). Si la raison du refus est quelque chose qui peut être résolu, alors vous pouvez [demander à nouveau la transaction](/help/dsp/inventory/on-demand-inventory-rerequest.md).

Si vous avez des questions sur le processus d’approbation, contactez l’équipe chargée de votre compte Adobe.
