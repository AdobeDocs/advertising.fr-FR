# Accepter une offre dans le [!UICONTROL Deal ID Inbox]

*Utilisateurs dans les comptes DSP mappés aux comptes SSP uniquement*

Utilisez le [!UICONTROL Deal ID inbox] pour accepter rapidement une transaction privée que vous avez déjà négociée avec un éditeur sur [!DNL FreeWheel], [!DNL Google Authorized Buyers] (anciennement appelé [!DNL AdX]) et [!DNL Magnite DV+] (anciennement [!DNL Rubicon]).

>[!NOTE]
>
>La configuration d’offres programmatiques garanties avec les éditeurs sur [!DNL FreeWheel] nécessite des autorisations et des étapes supplémentaires. Voir « [ Présentation de la configuration d’offres programmatiques garanties dans FreeWheel ](freewheel-overview.md) » pour plus d’informations.

1. Dans le menu principal, cliquez sur **[!UICONTROL Inventory]** > **[!UICONTROL Deals].**

1. Au-dessus de la liste des [!UICONTROL Deals], cliquez sur la barre bleue pour ouvrir la [!UICONTROL Deal ID inbox].

1. (Facultatif) Pour actualiser les détails de l’opération, cliquez sur **[!UICONTROL Refresh]**.

   DSP actualise automatiquement tous les détails de l’offre tous les jours à 4 :30 (heure de Paris). Elle actualise également toutes les offres [!DNL FreeWheel] et met à jour les offres existantes toutes les [!DNL Google] et [!DNL Magnite DV+] heures.

1. (Si vous avez précédemment ignoré l’offre) Cliquez sur l’onglet **[!UICONTROL Ignored Deals]** .

1. Pour consulter et confirmer les détails de l’opération, tels que l’éditeur et les dates, placez le curseur sur la ligne de l’opération et cliquez sur ![Vérifier](/help/dsp/assets/review.png).

1. Effectuez l’une des opérations suivantes :

   * Dans les détails de l’offre, cliquez sur **[!UICONTROL Accept]**.

   * Dans la [!UICONTROL Deal ID inbox], placez le curseur au-dessus de la ligne de l&#39;offre et cliquez sur ![Accepter](/help/dsp/assets/accept.png).

1. Dans les détails de l’offre :
   1. Renseignez les informations requises : **[!UICONTROL Publisher]**, **[!UICONTROL Media Type]** et **[!UICONTROL Deal Access]** (les annonceurs ayant accès à l’offre).
   1. (Facultatif) Spécifiez des comptes supplémentaires avec lesquels partager l&#39;opération ou joignez des libellés à l&#39;enregistrement de l&#39;opération.

   1. Cliquez sur **[!UICONTROL Save]**.

1. (Offres programmatiques garanties uniquement) Suivez les invites pour sélectionner une annonce (ou un pixel de suivi 1x1 pour les annonces gérées par l’éditeur) pour l’offre et créez un emplacement par défaut programmatique garanti qui cible l’offre.

Une fois que vous avez accepté une offre, elle est déplacée de la vue [!UICONTROL Deal ID inbox] vers la vue [!UICONTROL Inventory] > [!UICONTROL Deals] et l’offre est disponible en tant que source d’inventaire privée dans la section [!UICONTROL Inventory Targeting] de chaque emplacement.
