# Obtenir de l’aide

Vous pouvez envoyer une requête assistée par l’IA dans toute la documentation [!DNL DSP] et (annonceurs avec Advertising Creative) [!DNL Creative] ou ouvrir un guide d’utilisation complet pour [!DNL DSP]. Si la documentation ne répond pas à vos questions, contactez-nous.

Reportez-vous à la section « [Recherche de documentation du produit à l&#39;aide d&#39;une interface de chat assistée par IA](/help/dsp/agent-chat.md). »

## Contacter [!DNL Adobe]

Pour les problèmes de produit ou de compte, procédez comme suit :

* (Annonceurs avec des contrats en libre-service) Ouvrez un ticket pour votre organisation à l&#39;adresse [https://experienceleague.adobe.com/home?lang=fr#support](https://experienceleague.adobe.com/home?lang=fr&support-tab=home#support).

  Pour le produit, sélectionnez « [!UICONTROL Advertising - DSP] ».

* (Annonceurs avec des contrats de service complet) Contactez votre équipe de compte Adobe.

Dans le ticket ou la communication, indiquez les étapes détaillées pour reproduire le problème, les données pertinentes, le nom et la version du navigateur, ainsi que des informations sur le dépannage préliminaire que vous avez effectué. Pour les problèmes de rapport, ne supprimez pas le rapport et fournissez le nom du rapport et l’URL.
