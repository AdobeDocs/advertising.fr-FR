# Télécharger des données à partir de la vue [!UICONTROL Experiences]

Vous pouvez exporter des métadonnées sur les expériences de la vue [!UICONTROL Experiences] filtrée actuelle dans un fichier au format de feuille de calcul Microsoft Excel (XLSX).

Chaque ligne du fichier contient des détails sur une expérience, notamment : le nom et l’ID de l’expérience ; si l’expérience inclut ou non des annonces dynamiques ; le nombre de créatifs, d’offres groupées et de balises publicitaires associées ; si l’expérience inclut ou non un arbre de décision ; les tailles de création incluses ; les étiquettes applicables ; l’image miniature affichée par [!DNL Creative] pour l’expérience ; le nom des offres groupées attribuées ; et les noms des balises publicitaires associées.

1. Dans le menu principal, cliquez sur **[!UICONTROL Creative]** > **[!UICONTROL Experiences]**.

1. (Facultatif) [Personnaliser la vue](/help/creative/introduction/customize-data-views.md) pour inclure des expériences spécifiques.

1. Dans la barre d&#39;outils supérieure droite, cliquez sur ![Télécharger](/help/creative/assets/download.png "Télécharger").

   Le fichier est enregistré dans le dossier Téléchargements par défaut de votre navigateur.
